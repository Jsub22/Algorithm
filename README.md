## Git Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[PLATFORM]**|문제 해결|
|**[FIX]**|수정|
|**[DOCS]**|문서 (문서 추가, 수정, 삭제)|
|**[CHORE]**|빌드, 설정 파일|
